# Evidence archive contract

This directory is reserved for future, human-reviewed evidence from the public
demo: sanitized baseline output, final validation output, patches, summaries,
and run metadata.

GitHub Actions writes raw run material to its ephemeral runner first. Evidence
may be copied here only after secret scanning, privacy review, and removal of
machine-specific or sensitive values.

Never commit raw provider keys, raw runner logs, unsanitized model transcripts,
session files, or other secrets. If review is incomplete, keep the material out
of Git and record the hold separately.
